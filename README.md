# youtubeLiveChat取得

## 概要

youtubeライブのアーカイブからチャット一覧を取得する。

投稿日時、ユーザ名、コメントを抽出。

スーパーチャットをしていれば額も抽出。

取得結果はGoogleSpreadsheetに出力する。
